# Security Policy

## Reporting Security Issues

The K8up maintainers take security seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them through our [private vulnerability reporting](https://github.com/k8up-io/k8up/security/advisories/new) form.

For more details, see the [K8up security advisory page](https://github.com/k8up-io/k8up/security/advisories/new).
